# Diseño Web con Sass

Este proyecto es una pre-entrega enfocada en la implementación de Sass (SCSS) para el diseño y la estructuración de un sitio web. El objetivo es demostrar la capacidad de Sass para organizar y mantener el código CSS de manera eficiente.

## Descripción

El propósito principal de esta entrega es explorar y aplicar las funcionalidades de Sass, tales como variables, mixins, la modularización del código, etc, para crear un diseño web limpio y adaptable. Se ha puesto especial atención en la organización del código y la facilidad de mantenimiento, utilizando Sass para optimizar el flujo de trabajo de diseño.

## Características principales

* **Estructura modular:** Organización del código Sass en archivos separados para variables, mixins y estilos específicos.
* **Variables y mixins:** Uso de variables para mantener la consistencia de colores y tipografias, y mixins para reutilizar bloques de estilos.
* **Diseño adaptable:** Implementación de estilos que se adaptan a diferentes tamaños de pantalla.
* **Código limpio y legible:** Uso de la sintaxis de Sass para mejorar la claridad y el mantenimiento del código CSS.

## Tecnologías utilizadas

* HTML5
* Sass (SCSS)
* CSS3 (compilado desde Sass)
